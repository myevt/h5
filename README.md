# EVT 混合程序

## Usage

### 1、evt_getFungibleBalance
同质代币列表
[ "99990.00 YYA", "100000.00 YYB", "10000.00000 EVT" ]

### 2、evt_getOwnedTokens
非同质代币
[{domain:"ding.org", name:"token name"},...]

### 3、evt_getCreatedDomains
域列表
[{name:"ding.org"},...]

### 4、evt_getManagedGroups
组列表
[{name:"ding.org"},...]
